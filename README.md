## 📌 About This Fork

This is an enhanced version of the original [Cybersecurity-Mini-Projects](https://github.com/kienmarkdo/Cybersecurity-Mini-Projects) repository. I've made significant improvements, particularly to the keylogger project, including:

- Added encryption for stored logs
- Implemented legal compliance warnings
- Improved code structure and documentation
- Added time limits and secure email handling

## 📚 Project Overview

| Project | Description | Languages |
|---------|-------------|-----------|
